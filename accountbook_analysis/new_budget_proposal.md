## 현행 예산 문제 진단

### 1. 수입 < 예산
```
연간 수입:    ~62,000K
현행 예산 합계: 77,169K (내부경영 55.9M + 프로젝트 21.3M)
차이:         -15,169K ← 자산을 갈아먹는 구조
```

### 2. 저축/투자가 지출에 혼재
- 저축 5M + 주식 15M = 20M이 "내부경영 예산"에 포함
- 이것을 빼면 실제 소비 예산은 57.2M → 수입 62M 내에서 가능

### 3. 비정기 대형 지출 무방비
- 어머니 선물 7월 249만 (1건), 관계-선물 애플 53만 (1건)
- 자동차 유지 12월 389만 (보험 갱신 추정)
- 이런 지출이 월예산 페이스를 무의미하게 만듦

### 4. 식비 예산이 실적보다 낮음
- 예산 833K vs 실적 972K → 매달 적자 확정

## 새 예산안 설계 원칙

1. **수입 범위 내에서 설계** (연 62M 기준)
2. **저축/투자는 "남는 돈"으로** (지출 예산에 섞지 않음)
3. **고정/변동/프로젝트/비정기 4층 분리**
4. **비정기 대형 지출은 연간 버퍼로 관리**
5. **실적 기반 + 10% 버퍼**
