# 청지기 재정 시스템 (Steward Finance System)

## 핵심 철학

> "모든 소득은 하나님의 것이며, 나는 그것을 맡은 청지기다.
> 아끼는 것이 목표가 아니라, **맡겨진 것을 하나님의 뜻에 맞게 배분하는 것**이 목표다."

### 3대 원칙

1. **목표 주도 배분 (Mission-Driven Allocation)**
   - "줄여야 할 지출"이 아니라 "배정해야 할 사명"으로 접근
   - 각 예산 항목은 **왜 이 돈이 필요한가**에 대한 답이 있어야 함
   - 프로젝트(효도, 선교, 관계 등)가 예산의 중심

2. **연봉 내 설계, 부족 시 구함 (Design Within, Ask Beyond)**
   - 기본 프레임은 연간 수입 내에서 설계
   - 하나님의 일에 더 필요하면 → 더 벌거나 기도로 구함
   - "오버"가 아니라 "확장 요청"

3. **투명한 추적, 정직한 리뷰 (Transparent Tracking)**
   - 모든 지출은 자동으로 분류되고 예산 항목에 연결
   - 월간/연간 리뷰로 청지기직의 성실함을 점검
   - 데이터가 다음 기간 예산의 근거가 됨

## 예산 구조: 청지기 5층

> 기존 4층(고정→생활→프로젝트→비정기)을 **목적 우선순위**로 재편

```
┌─────────────────────────────────────────────────┐
│              하나님의 공급 (수입)                   │
│         월급 3.5M + 상여 17.5M = 연 60M           │
└──────────────────────┬──────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────┐
│  1층: 하나님의 몫 (First Fruits)                   │
│  헌금, 선교, 필리핀, 수련회                         │
│  → 가장 먼저 배정. 나머지를 쓰는 것이 아니라         │
│    먼저 드리고 나머지로 사는 것.                      │
└──────────────────────┬──────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────┐
│  2층: 사명 프로젝트 (Mission Projects)              │
│  효도, 관계, 찬양, 교회/양육, ACS, 경조사            │
│  → 하나님이 맡기신 관계와 섬김의 영역                 │
│    목표가 예산을 이끈다                              │
└──────────────────────┬──────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────┐
│  3층: 청지기 운영 (Stewardship Operations)          │
│  고정비(할부,보험,통신) + 생활비(식비,교통,생활)       │
│  → 청지기로서 건강하게 살기 위한 기본 비용             │
│    절약이 아니라 적정한 관리                          │
└──────────────────────┬──────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────┐
│  4층: 성장 씨앗 (Growth Seeds)                     │
│  저축, 투자, 자기계발(도서)                          │
│  → 미래의 청지기 역량을 키우는 투자                   │
│    상여금 중심으로 운용                              │
└──────────────────────┬──────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────┐
│  5층: 신뢰 여백 (Trust Margin)                     │
│  비정기(여행,패션,문화), 예비비, 대형이벤트            │
│  → 하나님이 주시는 여유와 돌발 상황 대비              │
│    "모자라면 구한다"의 실천 영역                      │
└─────────────────────────────────────────────────┘
```

### 기존(4층) vs 신규(5층) 비교

| 관점 | 기존 4층 | 신규 5층 |
|------|---------|---------|
| 분류 기준 | 통제 가능성 (고정→변동) | 목적 우선순위 (하나님→나) |
| 1순위 | 고정비 (할부, 보험) | 하나님의 몫 (헌금, 선교) |
| 프로젝트 위치 | 3층 (후순위) | 2층 (핵심) |
| 저축/투자 | 잔여금 처리 | 4층 성장 씨앗 (의도적) |
| 예비비 | 4층 세분화 | 5층 신뢰 여백 (여유) |
| 초과 시 대응 | 절감 | 확장 요청 (기도/추가소득) |

## 시스템 아키텍처

### 데이터 흐름

```
[뱅크샐러드]
     │ 내보내기 (CSV)
     ▼
[Gmail 수신]
     │ OpenClaw Pipeline (자동)
     ▼
┌─────────────────────────────────────┐
│         Google Sheets                │
│                                      │
│  ┌──────────────┐  ┌──────────────┐ │
│  │ 가계부 내역    │  │ 예산안 시트   │ │
│  │ (원장)        │←→│ (5층 구조)   │ │
│  │              │  │              │ │
│  │ A: 날짜      │  │ SUMIF로      │ │
│  │ G: 금액      │  │ K열 기준     │ │
│  │ J: 메모(프로젝트)│ │ 자동 집계    │ │
│  │ K: 상세(예산키) │  │              │ │
│  └──────────────┘  └──────────────┘ │
└──────────────────┬──────────────────┘
                   │
     ┌─────────────┼─────────────┐
     ▼             ▼             ▼
 [월간 리뷰]   [대시보드]   [알림/리포트]
 청지기 점검    실시간 현황   초과/부족 감지
```

### 7단계 OpenClaw Pipeline

```
1. ingest   → Gmail에서 뱅샐 메일 수신
2. unzip    → 첨부 ZIP 해제
3. normalize→ CSV 정규화
4. dedup    → 중복 제거
5. apply    → Google Sheets에 기록
6. categorize→ rules.json 기반 K열 자동 분류
7. budget   → 예산 대비 실적 연동 (신규 개발 대상)
```

### 핵심 연결고리: K열 (상세)

```
가계부 내역 K열 값  ←→  예산안 항목명
    "선교 헌금"    →   =SUMIF(K:K, "선교 헌금", G:G)
    "식비"         →   =SUMIF(K:K, "식비", G:G)
    "어머니 용돈"   →   =SUMIF(K:K, "어머니 용돈", G:G)
```

이 K열 태깅이 전체 시스템의 **단일 진실의 원천(Single Source of Truth)**.
태깅률이 곧 예산 추적 정확도.

## 월간 청지기 리뷰 체크리스트

1. **하나님의 몫**: 헌금/선교 예산 준수 여부
2. **사명 프로젝트**: 각 프로젝트 진행률 및 예산 소진율
3. **운영**: 고정비 이상 없는지, 생활비 적정 범위인지
4. **성장**: 저축/투자 목표 달성률
5. **여백**: 비정기 지출 발생 내역, 예비비 잔액
6. **K열 태깅률**: 미분류 거래 비율 → 다음 달 분류 규칙 보강
7. **감사/기도**: 이번 달 공급에 대한 감사, 다음 달 필요에 대한 기도
